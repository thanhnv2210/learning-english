# Prompt Templates

Below are templates to quickly create practice prompts.

1) PR Explanation (3 sentences)

Prompt:
"You are a senior engineer. I implemented X to fix Y. Write a concise PR description with: Motivation, What changed, How to test (3 steps). Keep it professional and 4-6 short sentences."

2) Standup (30s)

Prompt:
"Act as my standup coach. I will give 3 points: Yesterday, Today, Blockers. Make the sentence clear and natural for native English speakers. Keep it under 30s when spoken."

3) Code review suggestion (polite)

Prompt:
"Suggest a polite code review comment that recommends extracting a method for readability, with reasons and a short code example. Keep it 1-2 short paragraphs."

4) Speaking drill (60s)

Prompt:
"I will explain a bug fix for 60 seconds. Correct my grammar and phrasing and give 3 improvement tips, focusing on clarity for technical audiences." 

Store many variants and tag them for reuse.
