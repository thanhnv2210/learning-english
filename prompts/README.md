# Prompts — reusable practice prompts

Purpose: store short, copy-pastable prompts you can use to practice specific situations.

How to use

- Pick a prompt and either paste it into a chat-based practice tool or use it in a role-play with a colleague.
- Save variants with tags: `#speaking`, `#writing`, `#pr`, `#standup`.

Example files:
- `prompts/pr-explanation.md` — explaining a PR in 3-5 sentences.
- `prompts/speaking-60s.md` — 60s daily speak practice prompts.

Keep each prompt short (1–3 sentences) and include an example good answer.
