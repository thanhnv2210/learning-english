# Collocations — quick usage notes

Search CLI

- Run the simple search tool from the `collocations` folder:

```bash
python3 search.py --word deploy
python3 search.py --tag performance
```

Adding new entries

1. Copy `TEMPLATE.md` and fill the fields for the new collocation.
2. Add the markdown file to the folder (optional), and append the JSON entry to `data.json` if you want it searchable from the CLI.

Practice ideas

- Each week pick 5 collocations and make 3 spoken sentences using them.
- Use them in PR descriptions and daily standups; record 1 example per day.
