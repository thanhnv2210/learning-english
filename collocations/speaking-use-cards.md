# Speaking Use Cards — quick phrases for drills

These are short cards you can practice aloud (30–60s drills). Use them to build fluency for technical situations.

Card format: [Cue] — [One natural sentence]

- [deploy to production] — "We will deploy to production on Friday at 22:00 UTC."
- [optimize performance] — "We optimized query performance by adding an index and caching." 
- [fix a bug] — "I fixed a bug that caused null pointer exceptions in the payment flow." 
- [roll out to 10%] — "We'll roll out this change to 10% of users for a canary test." 
- [follow up with QA] — "I'll follow up with QA once the build passes to confirm test coverage."

Practice routine

- Pick 3 cards, record yourself saying each one, then listen and note 2 improvements.
- Replace literal translations with collocations; do not translate word-by-word.
