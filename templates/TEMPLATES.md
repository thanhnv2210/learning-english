# Templates — Markdown & PlantUML templates

This folder contains reusable templates for daily logs, monthly notes, issue reports, and PlantUML conversational flows.

Files

- `daily-template.md` — basic daily log template.
- `monthly-template.md` — monthly summary template.
- `issue-template.md` — structure for documenting an issue and corrections.
- `conversation-flow.puml` — PlantUML template for conversational flows (role-play), and `pr-process.puml` for PR process.

Usage

- Copy a template to the appropriate folder and fill it; keep the original template as canonical.
