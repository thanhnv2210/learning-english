# Monthly Notes — track progress every month

Naming: `YYYY-MM.md` (e.g., `2026-03.md`)

Structure (use `monthly-template.md` from templates):

- Summary: 3–5 bullet points of progress
- Recordings reviewed: list and short notes
- Repeated mistakes: list and plan to fix
- New phrases learned
- Demo / Presentation: topic and feedback
- Next month goals
